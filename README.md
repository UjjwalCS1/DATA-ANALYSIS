# 🎮 Video Game Sales Analysis

## 📌 Project Overview
This project focuses on analyzing video game sales data to uncover trends, patterns, and insights across different publishers, platforms, and regions. The analysis helps understand which factors contribute to a game's success.

---

## 📂 Dataset
The dataset contains information such as:
- Game Title
- Platform
- Year of Release
- Genre
- Publisher
- Sales (NA, EU, JP, Global)

---

## 🛠️ Technologies Used
- Python
- Pandas – Data manipulation
- NumPy – Numerical operations
- Matplotlib / Seaborn – Data visualization
- Jupyter Notebook

---

## 📊 Key Analysis Performed

### 1. Data Cleaning
- Handled missing values
- Fixed date parsing issues
- Removed duplicates

### 2. Exploratory Data Analysis (EDA)
- Top publishers by total sales
- Number of unique games per publisher
- Sales distribution by region
- Genre-wise performance

### 3. Visualizations
- Bar charts for top publishers
- Line plots for sales trends over time
- Pie charts for genre distribution
- Hidden Gems analysis

---

## 📈 Sample Insights
- Certain publishers dominate global sales
- Action and Sports genres perform consistently well
- Sales peaked during specific years
- Some lesser-known games show strong niche performance
